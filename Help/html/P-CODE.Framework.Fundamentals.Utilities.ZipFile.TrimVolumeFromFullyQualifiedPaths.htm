Field Value

true if [trim volume from fully qualified paths]; otherwise, false.

Remarks

when this is set, we trim the volume (eg C:) off any fully-qualified pathname, before writing the ZipEntry into the ZipFile. We default this to true. This allows Windows Explorer to read the zip archives properly.
