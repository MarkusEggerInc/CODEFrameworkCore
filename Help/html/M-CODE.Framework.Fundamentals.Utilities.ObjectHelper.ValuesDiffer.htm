Remarks

This method has been created to be easily able to compare objects of unknown types. In particular, this is useful when comparing two fields in a DataSet. This method can even handle byte arrays.

Examples

object o1 = "Hello"; object o2 = "World"; object o3 = 25; ObjectHelper.ValuesDiffer(o1, o2); // returns true; ObjectHelper.ValuesDiffer(o1, o3); // returns true;
